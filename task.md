## Задание 7

Разработать приложение для конфигурирования системного блока.
Системный блок представить в виде класса. 

Системный блок может содержать: корпус, материнскую плату, процессор, оперативную память, жесткий диск или SSD, видеокарту,
систему охлаждения.

Разработать метод, для добавления компонентов в системный блок. В текстовом виде сборщику задается вопрос, о том, какой
компонент будет установлен. В зависимости от ответа вызывается один из методов добавления компонентов. Каждый из методов
сообщает сборщику какой тип компонента необходимо установить(процессор, память и проч.), а сборщик вводит название или модель
компонента (Core i9, 64Gb Kingston, GeForce RTX 5060)


## Задание 8

Разработать приложение с использованием библиотеки **tkinter**.
Создать окно с двумя полями для ввода, подписями к ним и кнопкой.
Пользователь вводит два числа и нажимает на кнопку.
В том же окне пользователь получает результат в следующем виде: 

1. Если числа равны, то сообщение "Они равны"
2. Если числа не равны, то сообщение содержащее большее число

Для вывода сообщения использовать Label
